# HFF Develop — Landing Page

## Proyecto
Landing page para HFF Develop, consultora tech especializada en:
- Desarrollo de aplicaciones web
- Landings para visibilidad de negocios
- Consultoría e implementaciones de AI

## Stack
- HTML/CSS/JS puro (sin frameworks)
- Un solo archivo: index.html
- Deploy en Vercel

## Flujo de trabajo
- Rama de desarrollo: `develop`
- Nunca modificar `main` directamente
- Al terminar cambios importantes: merge de develop a main

## Estilo visual
El diseño está definido en index.html. Respetarlo siempre.
No cambiar paleta de colores, tipografías ni estructura general
salvo que se indique explícitamente.
